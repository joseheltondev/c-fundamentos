# C Fundamentos

Repositório com exercícios e projetos desenvolvidos durante meus estudos da linguagem C, com foco na prática de lógica de programação, estruturas de dados básicas e organização de código.

## Conteúdos Estudados

* Estruturas sequenciais
* Estruturas condicionais
* Estruturas de repetição
* Vetores
* Matrizes
* Funções
* Modularização de código (.c e .h)

## Projetos

### Conversor de Unidades

Aplicação de terminal capaz de converter:

* Comprimento (m, cm, mm, km)
* Massa (kg, g, mg, t)
* Temperatura (°C, °F, K)
* Tempo (s, min, h)

**Conceitos praticados:**

* Funções
* Switch Case
* Validação de entrada
* Modularização
* Organização de código em múltiplos arquivos

### Jokenpô

Jogo de terminal baseado em Pedra, Papel e Tesoura.

**Funcionalidades:**

* Escolha entre Pedra, Papel e Tesoura
* Geração aleatória da jogada do computador
* Validação de entradas inválidas
* Definição da quantidade de rodadas
* Placar de vitórias, derrotas e empates
* Cálculo da taxa de vitórias
* Exibição do vencedor da partida

**Conceitos praticados:**

* Estruturas condicionais
* Estruturas de repetição
* Geração de números aleatórios
* Entrada e saída de dados
* Validação de entrada

## Objetivo

Praticar lógica de programação e fortalecer os fundamentos da linguagem C por meio de exercícios e pequenos projetos, construindo uma base sólida para estudos mais avançados em desenvolvimento de software.
